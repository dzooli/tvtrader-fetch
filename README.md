# TvTrader-Fetch

[![SonarCloud analysis](https://github.com/dzooli/tvtrader-fetch/actions/workflows/sonarcloud.yml/badge.svg?branch=master)](https://github.com/dzooli/tvtrader-fetch/actions/workflows/sonarcloud.yml)

## Description

A REST service to fetch historical price data from TradingView.

Based on the [NestJs](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
